Privacy Layer

Local-first by design.

This page exists so privacy and storage behavior are explicit. It keeps the product feeling more complete without changing the way the core analysis works.

Guest Mode

Guests can run the scan and see a locked preview state. That preview is intentionally limited. It is there to show the reveal, not to provide the fully readable result.

Account Mode

Logged-in users unlock readable results, saved scans, and dashboard history. The value exchange is now much clearer across the whole site.

Storage

The site uses browser-local storage for session and saved scan data in the current build. This is why the product feels personal and immediate on the same device.

Pages Around The App

The new guide, playbook, roadmap, and privacy pages do not change the analysis engine. They add structure around it so the site feels like a larger product.
